# $addFont
It adds font to canvas.

# Usage
```
$addFont[canvas;font;path]
```

# Parameters 

---
Field|	Type| Description| Required
canvas|	string|	Name of the canvas.|	yes|
font|	string|	Font name.|	yes|
path|	string|	Path to font.|	yes|
---
